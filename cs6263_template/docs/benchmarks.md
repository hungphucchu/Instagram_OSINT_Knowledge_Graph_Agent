# Benchmarks

> Performance numbers from the team's own load test run. The TA reproduces
> this on their hardware during Phase 1 grading.

## Hardware Profile

- CPU: Intel x86_64, 8 cores
- Memory: 16 GB
- Disk: SSD, 50 GB free
- Network: 100 Mbit
- Container runtime: Docker 24.x
- Orchestration: Docker Compose v2

## Methodology

Load test driver: Locust 2.x (`tests/load/locustfile.py`).

Run command:

```bash
locust -f tests/load/locustfile.py \
  --headless -u 20 -r 5 -t 60s \
  --host=http://localhost:8080 \
  --csv=reports/loadtest
```

- `-u 20` — 20 concurrent users
- `-r 5` — ramp up at 5 users/second
- `-t 60s` — sustained load for 60 seconds
- Headline endpoint: `POST /api/query`

Rubric thresholds (Stress and Robustness category):
- ≥10 req/s sustained
- <5% error rate over 60 second window
